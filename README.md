# Exercise-1-SQL-Fundamentals.
This repository contains a collection of structured SQL practice exercises focused on querying and filtering data from an employees table. It covers essential SQL concepts such as SELECT, WHERE, DISTINCT, ORDER BY, LIMIT, and logical operators including AND, OR, and IN.

The exercises are designed to reinforce understanding of how to retrieve specific data, apply multiple conditions, and format queries correctly for real-world and exam scenarios. Each query follows proper SQL syntax and emphasizes readability through the use of uppercase keywords and clear clause structure.

This project demonstrates foundational data querying skills relevant to data analysis, business intelligence, and database management. It is suitable for beginners learning SQL as well as for revision and practice ahead of assessments.
